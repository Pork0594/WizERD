## What WizERD Does

WizERD parses PostgreSQL schema dumps (from `pg_dump` or `pg_dumpall`) and generates clean SVG ER diagrams with:

- Zero table overlap
- Smart edge routing via ELK layout
- 13 built-in themes
- Configurable spacing (compact/standard/spacious)
